# Eternel - 자연어 질의 기반 패널 데이터 검색 시스템

##  주요 기능

###  스마트 검색
- **자연어 질의**: "주로 즐겨하는 운동", "결혼 여부" 등 일반적인 언어로 검색
- **실시간 검색**: 백엔드 API와 연동하여 실제 패널 데이터 검색

###  인터페이스 모드
- **Test Search**: 간단한 검색창 인터페이스
- **AI Search**: 채팅 스타일의 대화형 검색

###  데이터 시각화
- **기본 차트**: 검색 결과의 기본적인 시각화
- **고급 차트**: 막대/선형/파이 차트 동시 표시 및 개별 선택
- **데이터 테이블**: 정렬, 페이지네이션, 통계 요약 기능
- **상세 결과**: 검색 결과의 상세 정보 표시

###  스마트 필터링
- **동적 필터**: 카테고리, 지역, 나이별 필터링
- **실시간 적용**: 필터 변경 시 즉시 결과 업데이트

###  AI 채팅 인터페이스
- **팝업 채팅**: 검색창 클릭으로 활성화되는 AI 어시스턴트
- **자연어 질문**: 데이터에 대한 자유로운 질문 가능
- **컨텍스트 유지**: 이전 대화 내용을 기반으로 한 연속 질문

## 🛠 기술 스택

### Frontend
- **React 18**: 컴포넌트 기반 UI 프레임워크
- **Tailwind CSS**: 유틸리티 우선 CSS 프레임워크
- **Chart.js**: 데이터 시각화 라이브러리
- **React Hooks**: 상태 관리 및 생명주기 관리

### Backend Integration
- **FastAPI**: Python 기반 백엔드 API
- **PostgreSQL**: 패널 데이터 저장소
- **RESTful API**: HTTP 기반 데이터 통신

### Design System
- **다크 테마**: 사이버펑크 스타일의 어두운 테마
- **네온 글로우**: 보라색 계열의 네온 효과
- **반응형 디자인**

##  설치 및 실행

### 1. 의존성 설치
```bash
cd Front_test
npm install
```

### 2. 환경 설정
```bash
# .env 파일 생성 (선택사항)
REACT_APP_API_BASE_URL=http://localhost:8000
REACT_APP_OPENAI_API_KEY=your_openai_api_key_here
```

### 3. 개발 서버 실행
```bash
npm start
```

애플리케이션이 http://localhost:3000 에서 실행됩니다.

### 4. 백엔드 연동 (선택사항)
백엔드 서버가 실행 중이어야 실제 데이터 검색이 가능합니다
```bash
# 백엔드 디렉토리에서
cd ../Data
source venv/bin/activate
cd backend
python main.py
```

##  사용 방법

### 기본 검색
1. 홈페이지에서 검색창에 키워드 입력
2. "체력", "결혼", "나이" 등 자연어로 질문
3. 검색 결과를 차트, 테이블, 상세보기로 확인

### 고급 기능
1. **필터 적용**: 지역, 나이, 카테고리별 필터링
2. **차트 모드 변경**: 기본/고급/테이블 뷰 전환
3. **AI 채팅**: 검색창 클릭 후 자연어로 질문
4. **데이터 내보내기**: 차트 이미지, PDF, CSV 다운로드

### 추천 검색어
-  체력 관리
-  결혼 상태
-  나이별 분석
-  지역별 특성
